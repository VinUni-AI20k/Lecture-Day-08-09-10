Chuan bi
Can tim hieu truoc / trong khi lam

Moi vai tro nen nam cac kien thuc tuong ung de lam nhanh trong 4 gio.

Khai niem RAG can hieu

  • Embedding la gi? Tai sao text thanh vector de search?
  • Dense vs Sparse retrieval: semantic search vs keyword match
  • Chunking: tai sao phai cat? Chunk lon/nho anh huong gi?
  • Grounded generation: ep LLM chi tra loi tu context
  • Hallucination: khi nao LLM "bia"? Tai sao nguy hiem?
  • Abstain: pipeline tu choi tra loi khi khong du data

Cong cu / thu vien can biet dung

  • ChromaDB: PersistentClient, get_or_create_collection, upsert, query
  • OpenAI API: embeddings.create() + chat.completions.create()
  • Hoac Gemini: genai.configure() + GenerativeModel.generate_content()
  • sentence-transformers: SentenceTransformer + CrossEncoder
  • rank-bm25: BM25Okapi de lam sparse search
  • python-dotenv: load API key tu .env

Ky nang Sprint 3 (chon 1 de sau)

  • Hybrid RRF: cong thuc Reciprocal Rank Fusion la gi?
  • Rerank: cross-encoder khac bi-encoder cho nao?
  • Query expansion: goi LLM de sinh synonym, alias
  • HyDE: Hypothetical Document Embedding — ý tuong
  • A/B testing: tai sao chi doi 1 bien moi lan?

Bay / loi thuong gap

  • Dung khac embedding model khi index vs khi query → sai hoan toan
  • Chunk cat giua dieu khoan → mat ngu canh → answer sai
  • Prompt khong co "only from context" → LLM tu bia → -50% diem cau
  • Quen metadata → khong co citation → mat diem Sprint 2
  • gq07 la bay: khong co data nhung van tra loi = penalty nang
  • Commit code sau 18:00 → khong duoc tinh
  • Report copy nhau → 0/40 cho TAT CA nguoi lien quan

Debug theo Error Tree (khi pipeline tra loi sai)

  • 1. Indexing? → list_chunks() — Chunk co dung? Metadata du?
  • 2. Retrieval? → score_context_recall() — Expected source co duoc retrieve?
  • 3. Generation? → score_faithfulness() — Answer co bam context? Prompt co "Answer only from context"?
